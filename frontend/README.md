# Frontend

This is the frontend of the Webb Hackathon 2026 project.

## Development with Docker

To start the development server, you need to have Docker and Docker Compose installed.

Then, run the following command in the `frontend` directory:

docker-compose up

The development server will be available at `http://localhost:5173`.

Changes to the source code will be automatically reloaded.

## Local Development (without Docker)

See guidelines.md

### Getting Started

npm install --legacy-peer-deps

npm run dev
